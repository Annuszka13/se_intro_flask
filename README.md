# Example App with Flask

## Development

1. Setup the virtualenv or use the existing:

   ```
   $ python3 -m venv .venv
   $ source .venv/bin/activate
   $ pip install -r requirements.txt
   ```

2. Run:

   ```
   $ export FLASK_APP=hello.py
   $ flask run
   ```
